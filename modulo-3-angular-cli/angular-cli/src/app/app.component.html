Comandos e exemplos do Angular CLI

Instalação e criação de projetos

  • Criar um novo projeto: ng new nome-do-projeto
  • Rodar o projeto no browser: ng serve

Criação de componentes, serviços e outros comandos

  • Criar um componente: ng generate component nome-componente ou ng g c nome-componente
  • Criar um serviço: ng generate service pasta-componente/nome-servico ou ng g s pasta-componente/nome-servico
  • Criar uma diretiva: ng generate directive minha-diretiva
  • Criar um pipe: ng generate pipe meu-pipe
  • Criar uma classe: ng generate class minha-classe
  • Criar uma interface: ng generate interface minha-interface
  • Criar um enum: ng generate enum meu-enum

Usando Sass, Less ou Stylus

  • Mudar a extensão padrão do estilo: ng config schematics.@schematics/angular:component.style scss
